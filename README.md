# swfrztool
**内核级希沃冰点还原管理工具**

## 概述
> 警告: 请在win10 x64及以上且安装希沃管家的环境中运行本程序 否则无法保证系统安全

本项目放弃使用传统的希沃自己的API 而是逆向`SWFreeze`、`SeewoKeLiteLady`等驱动 并在R0下修改冰点保护状态以及配置文件

实现了 **即时解除冰点还原保护而无需重启** **模拟冰点还原保护状态**

将来可能会实现 **保护白名单管理** **解析驱动位图选择性还原**...

## 使用说明
使用具有**管理员权限**的命令行启动程序

### config 修改冰点还原配置
> 注意: 在修改配置前先禁用状态模拟
```
swfrztool config [options] <drive1> <drive2>...
```
可以使用多种方案修改冰点保护配置 \(必须保护卷C 若不写则自动添加\)

`-r` 使用注入白名单的方式修改冰点配置 重启后生效

`-c` 使用修改驱动保护状态的方式修改配置 立即生效

`-x` 使用替换分发例程的方式修改冰点配置 立即生效 \(不推荐\)

### flt 模拟冰点还原状态
```
swfrztool flt <drive/off>...
```
集控会在关闭冰点还原一段时间后自动打开

程序通过劫持`SeewoKeLiteLady`的文件过滤回调重定向配置文件 所以使用前确保加载该驱动

即使是程序自身打开配置也会被重定向 因为 ~我懒得区分敌我~ 解析内核导出表会有问题添加新的API很麻烦

所以在修改配置前先禁用状态模拟

### help 获取帮助

## 示例
```bash
# 修改保护卷为C D 重启生效
swfrztool config -r C D
# 修改保护卷为不保护 立即生效
swfrztool config -c
# 模拟保护状态为保护卷C
swfrztool flt C
# 禁用状态模拟
swfrztool flt off
```

## 关于项目
本项目部分引用了R0提权项目`msrexec`和md5计算项目

除此之外 本项目所有代码 包括逆向和思路均有本人独立完成 与网络上其他有关希沃的项目无关

本项目使用的漏驱为公开的漏驱 若开启了更严格的签名校验可能无法加载 可以自己寻找能修改`msr寄存器`的驱动
